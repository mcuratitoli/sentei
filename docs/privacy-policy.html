Sentèi — Informativa sulla privacy

Ultimo aggiornamento: 25 giugno 2026

1. In sintesi

Sentèi è un'app per l'escursionismo. Non abbiamo server propri e non raccogliamo né conserviamo i tuoi dati personali su nostri sistemi. I dati che generi restano sul tuo dispositivo e, solo se scegli di attivare la sincronizzazione, nel tuo account cloud personale (iCloud o Google Drive), sotto il tuo controllo.

2. Posizione (GPS)

Con il tuo consenso, l'app usa la posizione del dispositivo solo sul dispositivo stesso per mostrarti sulla mappa e per le funzioni escursionistiche. L'accesso è limitato all'uso in primo piano (quando l'app è aperta). La tua posizione non viene trasmessa a noi né usata per profilazione o pubblicità. Puoi revocare il permesso in qualsiasi momento dalle impostazioni del sistema.

3. I tuoi tracciati e dati

I percorsi che crei (geometria, nome, distanza, dislivello, file GPX) sono salvati localmente sul dispositivo. Se attivi la sincronizzazione, gli stessi file vengono caricati nel tuo account cloud, in una cartella dedicata. Noi non vi abbiamo accesso.

  • iCloud Drive: i file restano nel tuo container iCloud personale, gestito da Apple.
  • Google Drive: se accedi col tuo account Google, l'app usa lo scope drive.file (accesso ai soli file creati dall'app). Puoi revocare l'accesso in qualsiasi momento dalle impostazioni del tuo account Google.

4. Servizi di terze parti

Per fornire mappe, sentieri ed elevazione, l'app contatta alcuni servizi esterni. Come per ogni connessione di rete, l'indirizzo IP del dispositivo è tecnicamente visibile al servizio contattato. Non condividiamo con essi alcun dato identificativo a fini di marketing.

  • Mapbox — mappe e terreno 3D; può raccogliere dati tecnici/telemetria d'uso secondo la propria informativa: mapbox.com/legal/privacy.
  • OpenStreetMap / Overpass API — rete dei sentieri e numeri CAI.
  • BRouter — calcolo del percorso lungo i sentieri.
  • Amazon AWS — tile di elevazione (Terrarium) per il calcolo del dislivello.

5. Cosa NON facciamo

Nessuna pubblicità, nessun tracciamento a fini di marketing, nessuna analitica di terze parti, nessuna vendita o cessione di dati personali.

6. Dati offline

Le aree che scarichi per l'uso offline (mappe ed elevazione) restano memorizzate sul tuo dispositivo.

7. Minori

L'app non è rivolta a minori di 13 anni e non raccoglie consapevolmente i loro dati personali.

8. Cancellazione dei dati

Puoi eliminare i singoli tracciati dall'app. I file sincronizzati si rimuovono dal tuo iCloud Drive o Google Drive. Disinstallando l'app rimuovi tutti i dati conservati localmente.

Eventuali aggiornamenti a questa informativa saranno pubblicati su questa pagina, con una nuova data.
